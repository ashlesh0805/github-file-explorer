# GitHub File Explorer

A simple web app to search and explore public GitHub repositories.  
Users can view repo details, browse folders, and preview files directly using the GitHub REST API.

## Features
- Search public repositories by name or topic.
- View stars, forks, and descriptions.
- Explore folders and files in real time.
- Preview file content inside the app.
- Clean glassmorphism design with HTML, CSS, and JavaScript.

## How to Run
1. Download or clone the project.
2. Open a terminal in the folder and run: npx serve 
Copy code
python -m http.server
Open the shown localhost link in your browser.

## Built With
HTML, CSS (Tailwind), JavaScript, and the GitHub REST API
